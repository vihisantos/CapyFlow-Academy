FAQ (Perguntas Frequentes)

Como recupero minha conta?

Como o sistema é Local-First, seus dados estão no seu navegador. Se limpar o cache sem backup, os dados somem. Use a função Exportar Save!

O certificado é válido?

O CapyFlow foca na habilidade prática. O certificado é simbólico para mostrar sua dedicação.

Posso criar meus próprios snippets?

Em breve! Estamos trabalhando no recurso 'Arena Creator'.

Precisa de ajuda humana?

Nossa equipe de capivaras (humanos) responde em até 24h úteis.
